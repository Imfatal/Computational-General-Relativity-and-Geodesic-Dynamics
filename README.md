# Computational-General-Relativity-and-Geodesic-Dynamics
Code to compute Christoffel symbols, geodesic equations, Ricci tensor, Ricci scalar, and Riemann tensor for diagonal gμν with rotational symmetry, and simulation of perihelion effect around a Schwarzschild black hole.

## Repository Structure

- `src/` : Python scripts for simulation and analytical calculations
- `figures/` : Output plots generated from simulations
- `report/` : Project report and LaTeX outputs
